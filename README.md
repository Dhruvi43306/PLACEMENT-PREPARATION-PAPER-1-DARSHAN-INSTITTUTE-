## 2. Leaders in an Array

Write a program to find the **leaders** in an array.

### Definition

A leader is an element that is greater than all elements to its right.

### Example

Input:

```text
16 17 4 3 5 2
```

Output:

```text
17 5 2
```

## 5. Ugly Number Check

Write a program to check whether a given number is an **Ugly Number**.

### Definition

An Ugly Number is a positive integer whose prime factors are only:

```text
2, 3, and 5
```

### Examples

- 14 → Not Ugly Number (contains factor 7)
- 6 → Ugly Number (2 × 3)

---

## 6. Pronic Numbers Between 1 and 100

Write a program to find all **Pronic Numbers** between 1 and 100.

### Definition

A Pronic Number is the product of two consecutive integers.

Formula:

```text
n = x × (x + 1)
```

### Examples

```text
2 = 1 × 2
6 = 2 × 3
12 = 3 × 4
20 = 4 × 5
```
